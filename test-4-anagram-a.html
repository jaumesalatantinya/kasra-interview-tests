Per cada paraula del array paraules busca els "anagrams"
"Anagrams" son dues paraules que tenen exactament les mateixes lletres amb ordre cambiat. Exemple "hola" i "ahol" son "anagrams"
exemple input: ["moo", "omo", "oom", "boo", "obo", "ooo"]
exemple output:
{
moo: ["omo", "oom"],
omo: ["moo", "oom"],
oom: ["moo", "omo"],
boo: ["obo"],
obo: ["boo"],
ooo: []
}
